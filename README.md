# auto-dns-switcher

One-click CLI to switch between anti-sanction DNS servers (Shecan, Electro, Radar, etc).

## Features
- 🚀 Fast and optimized
- 🔒 Secure by default
- 🛠 Easy to configure

## Installation
```bash
git clone https://github.com/HafezMosleh/auto-dns-switcher.git
cd auto-dns-switcher
```
