## 🌟 Overview
Say goodbye to complex network settings. Auto DNS Switcher is a seamless command-line interface that instantly routes your connection through trusted anti-sanction DNS providers (like Shecan, Electro, Radar). Perfect for developers and gamers looking for uninterrupted access.

## ✨ Key Features
- 🎯 **One-Click Execution**: Change DNS instantly without navigating menus.
- 🛡️ **Curated Providers**: Includes top-tier DNS servers for bypassing sanctions.
- 🔄 **Quick Reset**: Easily revert to Google/Cloudflare or clear your DNS cache.

## 🚀 Getting Started

### Prerequisites
Make sure you have `Python_Bash` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/auto-dns-switcher.git
cd auto-dns-switcher
chmod +x dns.py
```

### Usage
```bash
sudo python3 dns.py [shecan|electro|radar|clear]
```
